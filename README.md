## Quick start (local)
Prereqs: Node 18+, npm, (optional) Serverless Framework for deploys.

```bash
git clone <repo-url>
cd mini-cos
npm install
# Run backend in dev
cd backend && npm run dev
# Run frontend in dev
cd ../frontend && npm run dev
```
